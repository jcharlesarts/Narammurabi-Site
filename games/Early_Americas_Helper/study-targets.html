Quick reminders
Read the question carefully. The civilization name matters.
If it asks “why,” answer with because plus one reason.
For short answer, write one complete sentence.
Best strategy: say it out loud first, then check yourself.
Practice Set 1
Promised home location: describe the setting and name the lake. (Hint: water and island)
Geography: describe the landscape in two words.
Year-round water source: name the natural feature, also called cenotes.
Built to overcome barriers: name the network they built.
Emperor descended from gods: True or False, then add one reason that mattered.
Practice Set 2
Boys trained to be: ______ / Girls trained to work in the: ______.
Right below the emperor: priests and ______.
The empire grew because of: ______.
Priests performed human sacrifices: True or False, then explain why it mattered.
Women formed ______ through marriage and increased ______.
Practice Set 3
Decline: identify who conquered them and name the key city or year.
All boys served in the: ______.
Match each civilization to its farming method: Aztec, Maya, Inca.
Methods to use: chinampa, slash-and-burn, terrace farming.
All three kings were seen as all-powerful because: explain the belief.
Achievement sentence starters:
calendar system
writing system (glyphs)
astronomy
Quick check
Pick the best answer. Use this to see what you still need to review.
1) Which civilization built long road systems to connect a mountain empire? A) Aztec B) Inca C) Maya D) All three
2) Which farming method fits a swampy lake setting? A) terrace farming B) slash-and-burn C) chinampa D) dryland herding
3) Which clue pair best matches the Maya? A) roads + military service B) sinkholes + glyphs C) swampy island + lake D) terraces + Andes
4) Why were kings seen as all-powerful? A) elected by councils B) descended from the sun god C) won yearly contests D) chosen by farmers
5) Which civilization required all boys to serve in the military? A) Inca B) Maya C) Aztec D) none
6) Which capital sat on a lake island? A) Cuzco B) Tenochtitlan C) Tikal D) Machu Picchu
7) Which achievement is most associated with the Maya? A) terrace farming B) calendar and astronomy C) chinampas D) road networks
8) Which landscape best describes the Inca heartland? A) mountains and valleys B) flat river plains C) deserts D) swampy lowlands
Tip: point to the civilization name first, then answer.
